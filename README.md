# Mac-Record

> macOS 原生语音转写工具 — 录音、实时转文字、AI 整理纪要、语音输入法，一个 App 全搞定。

所有数据本地处理，隐私安全。基于 SwiftUI + SenseVoice 离线 ASR 引擎，Apple Silicon 原生运行。

## 核心能力

| 功能 | 说明 |
|------|------|
| 🎙️ **麦克风录音** | 一边录音一边自动出文字（SenseVoice 离线识别） |
| 🔊 **Self 记录** | 录电脑播放的声音（会议、网课、视频），基于 ScreenCaptureKit |
| ✨ **AI 纪要** | 对转写文字一键生成结构化会议纪要，支持超长录音自动分段+整合 |
| ⌨️ **语音输入** | 在任何 App 中长按热键说话，松开自动输入文字，AI 纠错 |

## 安装

### 方式一：下载 DMG（推荐）

前往 [Releases](https://github.com/hansluo/MAC-record/releases) 页面下载最新版 DMG，双击打开后拖入 Applications 即可。

> 已完成 Apple 公证（Notarization），可直接运行。

### 方式二：从源码构建

```bash
# 1. 克隆仓库
git clone https://github.com/hansluo/MAC-record.git
cd MAC-record

# 2. 下载依赖的大文件（不包含在 Git 中）
# - sherpa-onnx.xcframework (73MB) → 放入 Frameworks/
# - SenseVoice 模型文件 (228MB) → 放入 MacRecord/Resources/sensevoice-model/

# 3. 生成 Xcode 项目并构建
xcodegen generate
xcodebuild -scheme MacRecord build
```

**系统要求**：macOS 14.0+, Apple Silicon (M1/M2/M3/M4)

## 功能详解

### 🎙️ 麦克风录音 + 实时转写

**使用场景**：面对面会议、电话录音、采访、课堂笔记

1. 打开 Mac-Record，侧边栏底部点击 **红色录音按钮**
2. 录音开始后，侧边栏底部显示计时器和实时 ASR 文本预览
3. 支持 **暂停/恢复**
4. 说完后点「✓ 完成」

**亮点**：
- 边录边转，不用等录完再处理
- 内置音频增强（4阶 Butterworth 高通 + vDSP FFT 谱减法降噪 + 动态范围压缩）
- VAD 语音活动检测，自动分句
- 电脑休眠时自动暂停，唤醒后自动恢复并通知

### 🔊 Self 记录（录电脑声音）

**使用场景**：线上会议（腾讯会议/Zoom/飞书）、在线课程、看视频做笔记

1. 侧边栏底部切换音频源为「**Self 记录**」
2. 点击「开始 Self 记录」
3. 电脑播放的一切声音都会被录制并实时转写

> ⚠️ 首次使用需授权「录屏与系统录音」权限，授权后需完全退出 App 再重新打开。

### ✨ AI 智能纪要

**使用场景**：将大段录音文字整理成结构化会议纪要

1. 选中录音 → 详情页切换到「**AI 纪要**」Tab
2. 点击「生成 AI 纪要」
3. 超长录音自动分段处理 + 全局整合

**支持的 AI 服务商**：

| 服务商 | 推荐模型 | 特点 |
|--------|---------|------|
| DeepSeek | deepseek-chat | 中文效果好，价格实惠 |
| Google Gemini | gemini-2.5-flash | 速度快 |
| 通义千问 | qwen-plus | 国内访问稳定 |
| OpenAI | gpt-4o-mini | 综合能力强 |
| **本地模型** | Ollama / LM Studio / oMLX | 完全离线，零成本 |

**本地模型检测**：支持自定义端口 + API Key 输入，自动探测 Ollama (`/api/tags`) 和 OpenAI 兼容 (`/v1/models`) 接口。

### ⌨️ 语音输入法（全局可用）

**使用场景**：在微信、飞书、邮件、备忘录等任何输入框中用语音输入

1. 在任何 App 的输入框中，**长按 ⌥ Option 键**开始说话
2. 屏幕顶部出现浮窗显示「正在聆听…」
3. **松开按键**，自动识别 → AI 纠错 → 文字输入到光标位置

**智能纠错**：
- AI 自动纠正同音字错别字
- 自动添加标点符号
- 删除口头禅（长句时）
- 短句（10字以内）直接输出不走 AI，零延迟

### 📥 更多功能

- **导入外部音频**：支持 MP3 / M4A / WAV / AIFF，导入后自动转录
- **导出**：ASR 文本 / AI 纪要 / 音频文件 / 全部打包 ZIP
- **重新转录**：对已有录音用当前引擎重新识别
- **菜单栏常驻**：随时查看状态、一键开关语音输入

## 首次使用设置

### 1. 授权麦克风权限
首次录音时系统自动弹出请求 → 点击「好」

### 2. 授权辅助功能权限（语音输入必需）
系统设置 → 隐私与安全性 → 辅助功能 → 找到 Mac-Record 打开

### 3. 授权录屏权限（Self 记录必需）
系统设置 → 隐私与安全性 → 录屏与系统录音 → 打开 Mac-Record

### 4. 配置 AI 模型（可选）
设置 → AI 模型 → 选择「预设服务商」或「检测本地模型」→ 填入 API Key → 保存

### 5. 语音输入设置（可选）
设置 → 语音输入 → 选择触发热键（默认 ⌥ Option）→ 开启 LLM 纠错

## 常见问题

| 问题 | 解答 |
|------|------|
| 语音输入按键没反应？ | 检查辅助功能权限是否已授权 |
| Self 记录一直弹权限窗口？ | 授权后需**完全退出 App 再重新打开** |
| AI 纪要按钮灰色？ | 需先在设置中配置 AI 模型 |
| 支持哪些语言？ | 中文、英文、日文、韩文、粤语（自动检测） |
| 本地模型检测不到？ | 确认端口正确，oMLX 需填写 API Key |

## 技术架构

```
┌──────────────────────────────────────────────┐
│                  SwiftUI App                  │
│  ┌──────────┐  ┌──────────┐  ┌────────────┐  │
│  │ContentView│  │DetailView│  │SettingsView│  │
│  └────┬─────┘  └────┬─────┘  └─────┬──────┘  │
│       │              │              │          │
│  ┌────▼──────────────▼──────────────▼──────┐  │
│  │              AppState                    │  │
│  └────┬─────────┬──────────┬───────────┬───┘  │
│       │         │          │           │       │
│ ┌─────▼───┐ ┌──▼────┐ ┌───▼─────┐ ┌──▼────┐  │
│ │Audio    │ │Native │ │Summary  │ │Voice  │  │
│ │Recorder │ │ASR    │ │Generator│ │Input  │  │
│ │(AVAudio)│ │Service│ │(LLM)   │ │Service│  │
│ └─────────┘ └───┬───┘ └────┬────┘ └───────┘  │
│                 │          │                   │
│ ┌───────────────▼──┐  ┌───▼──────────────┐    │
│ │ SherpaOnnx (C)   │  │ LLMService       │    │
│ │ SenseVoice Model │  │ OpenAI Compatible │    │
│ │ Silero VAD       │  │ URL Auto-Normalize│    │
│ └──────────────────┘  └──────────────────┘    │
└──────────────────────────────────────────────┘
```

- **ASR 引擎**：SenseVoice Native（sherpa-onnx C API + Swift 封装），Silero VAD 实时语音活动检测
- **音频增强**：4阶 Butterworth 高通滤波 + vDSP FFT 谱减法降噪 + 动态范围压缩
- **系统音频**：ScreenCaptureKit 捕获
- **AI 纪要**：OpenAI 兼容协议，自动分段 + 递归整合，支持自定义 Prompt
- **休眠处理**：NSWorkspace sleep/wake 通知，自动暂停/恢复录音
