# Mac-Record 用户指南

> 一款 macOS 原生语音转写工具 — 录音、实时转文字、AI 整理纪要、语音输入法，一个 App 全搞定。

---

## 一、Mac-Record 能做什么？

Mac-Record 是一个**全能型语音工具**，安装在你的 Mac 上，所有数据都在本地处理，隐私安全。

它有 **4 大核心能力**：

| 功能 | 一句话说明 |
|------|-----------|
| 🎙️ 麦克风录音 | 一边录音一边自动出文字 |
| 🔊 Self 记录 | 录电脑播放的声音（会议、网课、视频） |
| ✨ AI 纪要 | 对转写文字一键生成结构化纪要 |
| ⌨️ 语音输入 | 在任何 App 中长按热键说话，松开自动输入文字 |

---

## 二、核心功能详解

### 1. 🎙️ 麦克风录音 + 实时转写

**使用场景**：面对面会议、电话录音、采访、课堂笔记

**怎么用**：
1. 打开 Mac-Record，侧边栏底部有一个 **红色录音按钮**
2. 点击「开始录音」
3. 屏幕右上角会弹出一个 **录音浮窗**，显示时长和实时文字
4. 说完后点「✓ 完成」

**亮点**：
- 边录边转，不用等录完再处理
- 自动命名（如「录音 4月9日 15:30」）
- 支持暂停 / 恢复
- 内置音频增强，弱麦克风信号也能识别

<!-- 📸 截图位置：侧边栏底部录音按钮 + 录音浮窗 -->

---

### 2. 🔊 Self 记录（录电脑声音）

**使用场景**：线上会议（腾讯会议/Zoom/飞书）、在线课程、看视频做笔记

**怎么用**：
1. 在侧边栏底部，把音频源从「麦克风」切换到「**Self 记录**」（蓝色图标）
2. 点击「开始 Self 记录」
3. 此时电脑播放的一切声音都会被录制并实时转写

**注意**：首次使用需要授权「录屏与系统录音」权限（后面会讲怎么设置）。

<!-- 📸 截图位置：麦克风 / Self 记录 切换器 -->

---

### 3. ✨ AI 智能纪要

**使用场景**：把一大段录音文字整理成结构化的会议纪要、笔记摘要

**怎么用**：
1. 选中一条录音，右侧详情页切换到「**AI 纪要**」Tab
2. 点击「生成 AI 纪要」按钮
3. 等待几秒，AI 会自动整理出带标题、分段的结构化纪要

**亮点**：
- 支持多种 AI 服务商（DeepSeek、Gemini、通义千问、OpenAI 等）
- 超长文本自动分段处理，不怕内容太多
- 纪要以 Markdown 格式渲染，清晰美观
- 每次生成保留历史，可以多次生成对比

**前提**：需要先在设置中配置 AI 模型（见下方「设置指南」）。

<!-- 📸 截图位置：AI 纪要 Tab + 生成按钮 + 纪要效果 -->

---

### 4. ⌨️ 语音输入法（全局可用）

**使用场景**：在微信、飞书、邮件、备忘录等任何输入框中用语音输入文字

**怎么用**：
1. 确保已开启语音输入（菜单栏图标 → 语音输入开关）
2. 在任何 App 的输入框中，**长按 ⌥ Option 键**开始说话
3. 屏幕顶部会出现一个浮窗，显示「正在聆听…」
4. **松开按键**，自动识别 → AI 纠错 → 文字输入到光标位置

**智能纠错**：
- AI 自动纠正同音字（如「扁奏穴位题」→「眼周穴位贴」）
- 自动添加标点符号
- 删除「嗯」「那个」等口头禅（长句时）
- 短句（10字以内）直接输出不走 AI，零延迟

**浮窗状态**：
| 状态 | 显示 |
|------|------|
| 正在聆听 | 🔴 脉冲红点 + 波形 |
| 识别中 | 🔍 文字 |
| 优化中 | ✨ AI 纠错 |
| 完成 | ✅ 显示最终文字 |

## 三、更多实用功能

### 📥 导入外部音频文件

**已有的录音文件也能用 Mac-Record 转文字！**

1. 点击工具栏的「**导入文件**」按钮（向下箭头图标）
2. 选择 MP3 / M4A / WAV / AIFF 格式的音频文件
3. 导入后自动开始转录
4. 转录完成后还可以点「生成 AI 纪要」整理内容

<!-- 📸 截图位置：工具栏导入按钮 + 导入后的录音详情 -->

### 📤 导出录音内容

选中一条录音后，点击工具栏「导出」按钮，支持 4 种导出格式：

| 格式 | 内容 |
|------|------|
| ASR 文本 (.txt) | 语音转写的原始文字 |
| AI 纪要 (.txt) | AI 生成的结构化纪要 |
| 音频文件 | 原始录音文件 |
| 全部 (.zip) | 文字 + 纪要 + 音频打包 |

<!-- 📸 截图位置：导出按钮 + 格式选择 -->

### 🔄 重新转录 & 补转录

- **重新转录**：对已有录音用当前 ASR 引擎重新识别（提升质量）
- **补转录**：导入的无文字录音，点击「补转录」按钮即可后补

### 📊 菜单栏常驻

Mac-Record 会在菜单栏显示一个小图标，随时可以：
- 查看语音输入状态（就绪 / 录音中 / 识别中）
- 一键开关语音输入
- 查看上次输入的文字
- 快速进入设置

## 四、首次使用设置指南

### Step 1：安装

1. 双击打开 `Mac-Record.dmg`
2. 将 Mac-Record 图标拖到右侧 Applications 文件夹
3. 打开「启动台」或「应用程序」文件夹，找到 Mac-Record 启动

> 💡 首次打开可能提示「来自身份不明的开发者」，在「系统设置 → 隐私与安全性」底部点击「仍要打开」即可。

<!-- 📸 截图位置：DMG 安装界面 -->

### Step 2：授权麦克风权限

首次录音时系统会自动弹出请求：

> "Mac-Record" 想要访问麦克风

→ 点击 **「好」** 即可

<!-- 📸 截图位置：麦克风权限弹窗 -->

### Step 3：授权辅助功能权限（语音输入必需）

语音输入功能需要「辅助功能」权限来检测热键和注入文字。

1. 打开 **系统设置** → **隐私与安全性** → **辅助功能**
2. 找到 **Mac-Record**，打开开关
3. 如果列表中没有，点击 `+` 号手动添加

> 💡 也可以在 Mac-Record 设置 → 语音输入 → 权限区域点击「前往授权」按钮自动跳转。

<!-- 📸 截图位置：辅助功能权限设置 -->

### Step 4：授权录屏权限（Self 记录必需）

「Self 记录」功能需要「录屏与系统录音」权限来捕获电脑声音。

1. 打开 **系统设置** → **隐私与安全性** → **录屏与系统录音**
2. 找到 **Mac-Record**，打开开关
3. **⚠️ 重要**：授权后必须 **完全退出 Mac-Record 再重新打开** 才会生效

> 如果开关已经打开但仍然不行，尝试 **关闭 → 重新打开** 开关，然后重启 App。

<!-- 📸 截图位置：录屏权限设置 -->

### Step 5：配置 AI 模型（可选 — 用于 AI 纪要和语音纠错）

如果你需要 AI 纪要或语音输入纠错功能，需要配置一个 AI 模型。

1. 打开 **设置** → **AI 模型** Tab
2. 点击 **「预设服务商」** 菜单，选择你使用的服务
3. 填入你的 **API Key**
4. 点击 **「测试连接」** 确认可用
5. 点击 **「保存」**

**支持的服务商**：

| 服务商 | 推荐模型 | 特点 |
|--------|---------|------|
| DeepSeek | deepseek-chat | 中文效果好，价格实惠 |
| Google Gemini | gemini-2.5-flash | 速度快 |
| 通义千问 | qwen-plus | 阿里云，国内访问稳定 |
| OpenAI | gpt-4o-mini | 综合能力强 |
| MiniMax | MiniMax-Text-01 | 性价比高 |

> 💡 也支持本地模型！如果你安装了 Ollama 或 LM Studio，点击「检测本地模型」会自动发现。

<!-- 📸 截图位置：AI 模型配置页面 + 预设菜单 -->

### Step 6：语音输入设置（可选）

1. 打开 **设置** → **语音输入** Tab
2. 确认「启用语音输入」开关已打开
3. 选择触发热键（默认是 **⌥ Option**，也可以选 Command、Control、Fn）
4. 调节长按阈值（默认 400ms，太灵敏就调大，反应慢就调小）
5. 开启 **LLM 纠错**（推荐，需要已配置 AI 模型）

## 五、常见问题

| 问题 | 解答 |
|------|------|
| 语音输入按键没反应？ | 检查辅助功能权限是否已授权（设置 → 隐私与安全性 → 辅助功能） |
| Self 记录一直弹权限窗口？ | 授权录屏权限后需**完全退出 App 再重新打开**才能生效 |
| AI 纪要按钮灰色？ | 需要先在设置中配置 AI 模型并确保有文字内容 |
| 支持哪些语言？ | 中文、英文、日文、韩文、粤语（自动检测，无需手动选择） |
| 已有的音频能转文字吗？ | 能！点工具栏「导入文件」按钮，支持 MP3/M4A/WAV/AIFF，导入后自动转写，再点「AI 纪要」即可整理 |
| 怎么导出内容？ | 选中录音 → 工具栏「导出」按钮 → 可选导出文字、纪要、音频或全部打包 ZIP |
| 语音输入太灵敏/太迟钝？ | 设置 → 语音输入 → 调节「长按阈值」滑块（200~800ms） |
